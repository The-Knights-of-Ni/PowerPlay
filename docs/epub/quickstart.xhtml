Quickstart

Building the Code

This guide will help you configure the IDE/Environment.

External Tools:

  1. Android Studio

  2. Gradle

  3. Git

  4. IntelliJ Ultimate (Optional)

To Build the code:

  1. Build the gradle project using the build gui

  2. Use adb to connect to the robot via
    command

    adb connect 192.168.43.1:5555

  3. Run the code
