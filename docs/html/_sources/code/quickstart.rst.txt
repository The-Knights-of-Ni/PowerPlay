Quickstart
===========

Building the Code
___________________
This guide will help you configure the IDE/Environment.

External Tools:

1. Android Studio
2. Gradle
3. Git
4. IntelliJ Ultimate (Optional)


Use git to clone the project

.. code:: shell

    git clone https://github.com/The-Knights-of-Ni/FreightFrenzy

To Build the gradle project use the build gui (Ctrl+F9 on windows)
Use adb to connect to the robot via

.. code:: shell

    adb connect 192.168.43.1:5555

Run the code (Shift+F10)
